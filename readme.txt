Tested up to: 2.8.4
Stable tag: 0.1

== Description ==
This plugin detects if your site is being viewed by iPhone (or iPod) and switches to an iPhone / iPod  theme.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select the a iPhone compatible theme in the plugin options panel
