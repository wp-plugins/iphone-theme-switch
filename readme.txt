== Description ==
This plugin detects if your site is being viewed by iPhone (or iPod) and switches to an iPhone / iPod  theme. iUI iPhone theme included.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` folder
2. Upload 'iUI' theme to the '/wp-content/themes/' folder
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Select the iUI theme (or a different one suitable for iPhone / iPod) in the iPhone theme switch panel under the admin setting menu.

== Frequently Asked Questions ==

= Where are my posts? =
If you have selected a page for your posts, for example 'News' in the 'readers' settings panel: A Static page: posts page: 'News' your posts will be under the 'News' button.
If you don't have a special page for your blog or home a button will be created called 'home'
